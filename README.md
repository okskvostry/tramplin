# Трамплин — платформа для поиска работы студентами

## Описание проекта

«Трамплин» — это экосистема, где студенты и выпускники могут найти стажировку, вакансию или ментора, а работодатели — разместить предложения и найти талантливых кандидатов. Платформа включает в себя:

- Регистрацию и авторизацию для трех ролей: соискатель, работодатель, куратор
- Личные кабинеты с возможностью редактирования профиля
- Создание и публикацию возможностей (вакансии, стажировки, мероприятия, менторские программы)
- Загрузку аватаров, логотипов и фотографий

## Технологии

- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Backend**: PHP 8.2
- **База данных**: MySQL 8.0
- **Контейнеризация**: Docker, Docker Compose

## Требования

Для запуска проекта необходимо установить:

- [Docker Desktop](https://www.docker.com/products/docker-desktop/)
- [Podman](https://podman.io/) (альтернатива, если Docker недоступен)

## Быстрый старт

### 1. Скачайте и разархивируйте репозиторий
### 2. Через командную строку зайдите в папку проекта (нужен весь путь от диска C до места разархивации):
cd tramplin

### 3. Запустите через командную строку:
docker-compose up -d

(или podman-compose up -d)

### 4. Пользуйтесь!
http://localhost:8080 — главная страница

http://localhost:8081 — phpMyAdmin (логин: root, пароль: root)
